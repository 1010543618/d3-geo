## Installing

If you use NPM, `npm install d3-geo`. Otherwise, download the [latest release](https://github.com/d3/d3-geo/releases/latest). You can also load directly from [d3js.org](https://d3js.org), either as a [standalone library](https://d3js.org/d3-geo.v0.0.min.js) or as part of [D3 4.0 alpha](https://github.com/mbostock/d3/tree/4). AMD, CommonJS, and vanilla environments are supported. In vanilla, a `d3_geo` global is exported:

```html
<script src="https://d3js.org/d3-geo.v0.0.min.js"></script>
<script>

var stream = d3_geo.geoStream();

</script>
```

[Try d3-geo in your browser.](https://tonicdev.com/npm/d3-geo)
